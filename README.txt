KYOTO DAYS　提出用サイト説明

【基本情報】
1. ホームページ：index.html
2. HTMLページ数：22ページ
3. テーマ：京都観光と留学生生活
4. 使用技術：HTML5 / CSS3 / JavaScript
5. CSS、JavaScript、画像はすべて相対パスで読み込んでいます。
6. 外部CDNや外部JavaScriptライブラリには依存せず、オフラインで閲覧できます。

【デザイン】
・京都の案内冊子を意識した、落ち着いた編集デザインです。
・明朝体の見出し、朱色と苔色、細い罫線、紙面風の余白を使用しています。
・観光地、交通、食文化の実景写真11点をローカル保存しています。
・写真の作者、ライセンス、出典は「写真・画像出典.txt」に記載しています。

【JavaScript機能】
・スマートフォン用ナビゲーション
・観光スポットのカテゴリ絞り込み
・写真ギャラリーのライトボックス
・京都知識クイズ
・お問い合わせフォームの入力チェック
・FAQアコーディオン
・ページ上部へ戻るボタン

【提出前に行うこと】
・「小组成员分工明细.txt」に実際の氏名と学籍番号を記入してください。
・実際のグループ番号が第41組でない場合は、フォルダ名とZIP名を変更してください。


【デザイン改訂】
平安・町家・古典意匠を現代の編集デザインとして再構成。青海波、和紙調背景、弁柄・苔・群青の配色、スクロール表示、雲の移動、画像の緩やかなドリフトを追加。
